## 🔎 What is pwgen-pwa?

pwgen-pwa is a password generator that works right in your web browser. It creates strong, secure passwords without sending your data over the internet. Since it runs offline, you can trust it to keep your passwords safe. It uses modern browser features to save your settings and work even when you are not connected to the internet.

This app works on Windows and other platforms that support modern browsers. You do not need an internet connection after you open it once. It runs fast and uses standard web technologies.
